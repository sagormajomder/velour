# VELOUR — Premium E-Commerce Storefront

VELOUR is a high-end, production-grade e-commerce storefront for traditional and contemporary South-Asian fashion. Blending premium cultural heritage (Panjabi, Saree, Kurti) with modern styles (Polo shirts), it features elegant matte black and gold accents, fluid animations, and complete client-side interactions.

## Key Features

- 🌟 **Design System**: A sleek, premium color palette (matte black primary, gold accent, and warm off-white secondary backgrounds) with full dark mode support.
- 🗂️ **Curated Category Showcase**: Responsive editorial layout showcasing distinct collections (Panjabi, Saree, Kurti, Polo Shirt) with smooth glassmorphic zooms and button reveals on hover.
- 🔍 **Instant Client-Side Search**: Search by product name, category, or description with instant results rendering.
- 🏷️ **Tab & Pill Category Filters**: Filter products dynamically by categories with a smooth sliding pill background animation.
- ↕️ **Advanced Sorting Options**: Sort products instantly by price (Low to High, High to Low), name (A-Z, Z-A), and customer rating.
- 🛒 **Interactive Cart Context**: Full shopping cart functionality (Add, Remove, Quantity adjustments, and Subtotal calculations) with browser `localStorage` persistence.

## Technology Used

- **Frontend**: TypeScript, Next.js
- **Styling**: TailwindCSS (v4), Shadcn, Motion
- **Deployment**: Vercel

## Run it Locally

Please follow the below instructions to run this project in your machine:

1. Clone this repository
   ```sh
   git clone https://github.com/sagormajomder/velour.git
   ```
2. Open the directory "velour" into visual studio code
3. Open Terminal and run `npm i` or `pnpm install` to install all dependencies
4. Run `npm run dev` or `pnpm dev` to run the project into browser.
   The project will be available on http://localhost:3000/ by default.
